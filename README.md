# Lecture Notes Viewer

Upload PDF slides + a transcript, get back beautifully rendered Markdown lecture notes (and PDFs) generated by Claude. Runs locally on your Mac.

## Install on a new Mac

You'll need: macOS, Python 3.11+, and an Anthropic API key from https://console.anthropic.com/settings/keys.

```bash
# 1. Clone the repo somewhere convenient (e.g. your home folder)
git clone https://github.com/<your-username>/lecture-notes-viewer.git ~/lecture-notes-viewer
cd ~/lecture-notes-viewer

# 2. Put the launcher on your Desktop
ln -s "$PWD/Launch Lecture Notes.command" ~/Desktop/
```

That's it. **Double-click `Launch Lecture Notes.command` on your Desktop.**

The first launch will:
- Create a Python virtualenv and install dependencies (~1 minute)
- Download Chromium for PDF export (~150 MB, one-time)
- Prompt you in the Terminal window for your Anthropic API key
- Open the app in your browser at http://localhost:8080

Every subsequent launch just runs the server.

To stop the server, press **Ctrl-C** in the Terminal window (or just close it).

## If Python is missing

If you don't have Python 3.11+, the launcher will tell you. Install via either:

```bash
xcode-select --install            # easiest
# OR
brew install python               # if you have Homebrew
```

Then re-launch.

## Updating

Once you've cloned, pull updates with:

```bash
cd ~/lecture-notes-viewer && git pull
```

The Desktop launcher always points at the latest version.
